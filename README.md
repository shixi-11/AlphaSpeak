## 🚀 部署

### Render 部署

1. 连接 GitHub 仓库
2. 设置环境变量：
   - `BOT_TOKEN` - Telegram Bot API Token
   - `TTS_ENABLED` - 是否启用语音 (true/false)
3. Build Command: `pip install -r requirements.txt`
4. Start Command: `gunicorn webhook:app`

### 阿里云部署

```bash
# SSH 登录服务器
ssh root@your-server-ip

# 执行部署脚本
bash deploy.sh
```
